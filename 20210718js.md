## ES2015+

var 쓰지말것 => const, let으로 대체 !

var는 함수 스코프이고 const, let은 블록 스코프이다

즉, 블록단위로 스코프를 관리해주는게 코드 관리측면에서 유리하다.

const = 말그대로 const, 불변 값
let = 일반적 변수 선언
