# 웹의 특징 

- WWW = World Wide Web: 팀 버너스리에 의해 개발되었다.
 
- 웹과 인터넷은 다르다.
 
- 웹의 3요소는 HTTP, HTML, URL이다.

## HTTP = HyperText Transfer Protocol

WWW 상에서 정보를 주고받을 수 있는 프로토콜

TCP와 UDP를 사용하며, 80번 포트 사용

## HTTP의 특징

비교적 간단하며 확장 가능하다.

상태가 없다(= stateless).
 
HTTP1 --> HTTP/1.1 --> HTTP/2 로 버전이 변경되었으며, 계속 발전하는 중이다.
